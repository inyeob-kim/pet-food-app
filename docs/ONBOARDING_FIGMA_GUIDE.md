## 1. 전체 플로우 개요

### 플로우 다이어그램

```
[앱 시작]
    ↓
┌─────────────────────────┐
│ Step 1: 닉네임          │ ← 2~12자 입력
└───────────┬─────────────┘
            ↓
┌─────────────────────────┐
│ Step 2: 아이 이름       │ ← 1~20자 입력
└───────────┬─────────────┘
            ↓
┌─────────────────────────┐
│ Step 3: 종 선택         │ ← 강아지 🐶 / 고양이 🐱
└───────────┬─────────────┘
            ↓
┌─────────────────────────┐
│ Step 4: 나이 정보       │ ← 생년월일 📅 / 대략 나이 🎈
└───────────┬─────────────┘
            ↓
        ┌───┴───┐
        │       │
    [강아지] [고양이]
        │       │
        ↓       ↓
┌──────────┐   ┌──────────┐
│ Step 5:  │   │ Step 6:  │
│ 품종 선택│   │ 성별+중성│
└────┬────┘   └────┬────┘
     │             │
     └──────┬──────┘
            ↓
┌─────────────────────────┐
│ Step 6/7: 성별+중성화   │ ← 남/여, 예/아니오/모름
└───────────┬─────────────┘
            ↓
┌─────────────────────────┐
│ Step 7/8: 몸무게        │ ← 0.1~99.9kg
└───────────┬─────────────┘
            ↓
┌─────────────────────────┐
│ Step 8/9: 비만도 (BCS)  │ ← 1~9 슬라이더
└───────────┬─────────────┘
            ↓
┌─────────────────────────┐
│ Step 9/10: 건강 고민    │ ← 다중 선택 칩
└───────────┬─────────────┘
            ↓
┌─────────────────────────┐
│ Step 10/11: 음식 알레르기│ ← 다중 선택 칩

            ↓

            ↓
    [헤이제노 시작하기]
            ↓
    [메인 홈 화면]
```

**총 단계**: 10~12단계 (종에 따라 다름)
- 강아지: 12단계
- 고양이: 11단계 (품종 선택 스킵)

## 2. 디자인 시스템 (토스 스타일)

### 색상 팔레트

| 색상 | Hex | 용도 |
|------|-----|------|
| **Primary** | `#2563EB` | CTA 버튼, 선택 상태, 강조 |
| **Text Main** | `#111827` | 제목, 본문 텍스트 |
| **Text Sub** | `#6B7280` | 보조 텍스트, 힌트 |
| **Positive** | `#16A34A` | 성공, 건강 상태 |
| **Negative** | `#EF4444` | 에러, 경고 |
| **Background** | `#F7F8FA` | 화면 배경 |
| **Surface** | `#FFFFFF` | 카드, 입력 필드 배경 |
| **Divider** | `#E5E7EB` | 구분선 (최소 사용) |

### 타이포그래피

| 스타일 | 크기 | 굵기 | 용도 | 색상 |
|--------|------|------|------|------|
| **Hero** | 38px | Bold (700) | 포인트 숫자 | Text Main |
| **Title** | 23px | SemiBold (600) | 화면 제목 | Text Main |
| **Body** | 17px | Medium (500) | 본문, 입력값 | Text Main |
| **Sub** | 14px | Regular (400) | 보조 텍스트, 힌트 | Text Sub |

**폰트**: Pretendard (fallback: NotoSansKR, Apple SD Gothic Neo, Roboto)
**Letter Spacing**: -0.3 (기본), -0.5 (Hero), -0.4 (Title)
**Line Height**: 1.4 (기본), 1.2 (Hero), 1.3 (Title)

### 레이아웃 규칙

- **화면 패딩**: 좌우 16px, 상하 16~24px
- **섹션 간격**: 24~32px
- **컴포넌트 간격**: 12~16px
- **카드/구분선 금지**: 토스 스타일 원칙
- **여백으로만 구분**: spacing으로 정보 위계 표현

### 버튼 스펙

| 버튼 타입 | 높이 | Border Radius | 패딩 | 텍스트 스타일 |
|-----------|------|---------------|------|---------------|
| **Primary CTA** | 52~56px | 16~18px | 좌우 24px | Body, Bold, White |
| **Secondary** | 36px | 12~14px | 좌우 16px | Sub, Medium |
| **Text Link** | Auto | - | - | Sub, Primary, Underline |

## 3. 화면별 상세 스펙

### 공통 레이아웃 구조

```
┌─────────────────────────────┐
│ [진행률 바] [뒤로가기]      │ ← 상단 고정 (SafeArea)
├─────────────────────────────┤
│                             │
│   [큰 이모지 아이콘]        │ ← 80x80px
│                             │
│   [제목 텍스트]             │ ← Title 스타일
│   [서브멘트 텍스트]         │ ← Sub 스타일 (선택)
│                             │
│   [컨텐츠 영역]             │ ← ScrollView
│   (입력/선택 UI)            │
│                             │
│                             │
├─────────────────────────────┤
│ [CTA 버튼]                  │ ← 하단 고정
└─────────────────────────────┘
```

### Step 1: 닉네임 입력

**레이아웃**:
- 진행률: 1/10 (10%)
- 뒤로가기: 없음 (첫 화면)

**컨텐츠**:
```
[이모지] 😊 (80x80)

[제목] "안녕하세요 😊"
[서브] "헤이제노에서 쓸 닉네임만 먼저 정해볼까요?"

[입력 필드]
- 높이: 56px
- Border Radius: 12px
- 배경: Surface (#FFFFFF)
- Border: 1px, Divider 색상
- Placeholder: "닉네임을 입력해주세요" (Sub 스타일)
- Text: Body 스타일
- Counter: 우측 하단, Sub 스타일, "{length}/12"

[추천 버튼] (선택)
- 높이: 36px
- Border Radius: 12px
- 배경: Primary Soft (#EFF6FF)
- 텍스트: "🎲 추천받기" (Sub, Primary 색상)
```

**CTA 버튼**:
- 텍스트: "다음"
- 비활성화: nickname.length < 2
- 활성화: nickname.length >= 2

## 5. 인터랙션 & 애니메이션

### 화면 전환

- **전환 방식**: FadeTransition 또는 Hero 애니메이션
- **Duration**: 300ms
- **Curve**: easeInOut

### 선택 피드백

- **Haptic Feedback**: 가볍게 (Light)
- **애니메이션**: Scale 1.0 → 0.95 → 1.0 (150ms)

### BCS 캐릭터 애니메이션

- **슬라이더 변경 시**: 실루엣 크기 부드럽게 변화 (300ms)
- **첫 선택 시**: 하트 팝업 (600ms)
  - Scale: 0 → 1.2 → 1.0
  - Opacity: 1 → 0
  - 위치: 캐릭터 상단

### 입력 필드

- **Focus 시**: Border 색상 Primary로 변경 (200ms)
- **에러 시**: Border 색상 Negative로 변경 + 에러 메시지 표시

### 버튼

- **비활성화**: Opacity 0.5, 터치 불가
- **활성화**: Opacity 1.0, Primary 배경
- **터치 시**: Scale 0.98 (100ms)
