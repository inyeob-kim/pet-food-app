# 앱 전체 화면별 와이어프레임 및 기능 정리

## 📱 화면 구조 개요

### 네비게이션 구조
```
앱 시작
  ↓
[초기 스플래시] → 온보딩 완료 여부 확인
  ├─ 미완료 → [온보딩 플로우]
  └─ 완료 → [홈 화면]
      ↓
[바텀 탭 네비게이션]
  ├─ 홈 (/home)
  ├─ 찜한 사료 (/watch)
  ├─ 사료마켓 (/market)
  ├─ 혜택 (/benefits)
  └─ 더보기 (/me)
```

### 1.2 온보딩 플로우 (`OnboardingFlowV2`)
**경로**: `/onboarding` 또는 `/onboarding_v2?mode=add_pet`  
**파일**: `frontend/lib/onboarding_v2/onboarding_flow.dart`

**와이어프레임**:
```
┌─────────────────────────┐
│  [←]  [진행률 바]       │
│                         │
│   [현재 단계 콘텐츠]     │
│                         │
│                         │
│   [다음 버튼]            │
└─────────────────────────┘
```

**단계별 구성** (총 11단계):

#### Step 1: 닉네임 입력
- **컨텐츠**: 텍스트 입력 필드
- **기능**: 사용자 닉네임 입력

#### Step 2: 펫 이름 입력
- **컨텐츠**: 텍스트 입력 필드
- **기능**: 반려동물 이름 입력

#### Step 3: 종류 선택
- **컨텐츠**: 강아지/고양이 선택 카드
- **기능**: 종류 선택 (DOG/CAT)

#### Step 4: 나이 입력
- **컨텐츠**: 나이 입력 필드 또는 단계 선택
- **기능**: 나이 또는 나이 단계 선택 (PUPPY/ADULT/SENIOR)

#### Step 5: 품종 선택
- **컨텐츠**: 품종 선택 카드 그리드
- **기능**: 품종 코드 선택

#### Step 6: 성별 및 중성화
- **컨텐츠**: 성별 선택 + 중성화 여부 토글
- **기능**: 성별 및 중성화 상태 선택

#### Step 7: 체중 입력
- **컨텐츠**: 체중 입력 필드 (kg)
- **기능**: 체중 입력

#### Step 8: BCS (Body Condition Score)
- **컨텐츠**: 슬라이더 (1-9점)
- **기능**: 체형 점수 선택

#### Step 9: 건강 고민 선택
- **컨텐츠**: 건강 고민 태그 선택 (다중 선택)
- **기능**: 건강 고민 리스트 선택 (예: 비만, 관절, 피부 등)

#### Step 10: 알레르기 선택
- **컨텐츠**: 알레르기 항목 선택 (다중 선택) + 기타 알레르기 텍스트 입력
- **기능**: 음식 알레르기 및 기타 알레르기 입력

#### Step 11: 사진 업로드
- **컨텐츠**: 사진 선택/촬영 버튼
- **기능**: 갤러리에서 선택 또는 카메라로 촬영

**공통 기능**:
- 진행률 바 표시
- 이전/다음 버튼
- 스킵 가능 (일부 단계)
- 데이터 저장 및 API 전송

### 2.1 홈 화면 (`HomeScreen`)
**경로**: `/home`  
**파일**: `frontend/lib/features/home/presentation/screens/home_screen.dart`

**와이어프레임**:
```
┌─────────────────────────┐
│ 안녕하세요, {닉네임}님! │
├─────────────────────────┤
│                         │
│ [펫 프로필 카드]         │
│ - 펫 아바타              │
│ - 이름, 종류, 나이       │
│ - 건강 고민 태그         │
│ - 알레르기 정보          │
│ - 프로필 수정 버튼       │
│                         │
│ [현재 급여 사료 카드]    │
│ - 사료명                 │
│ - 현재 최저가            │
│ - 가격 변동              │
│ - 알림 설정              │
│                         │
│ [추천 카드]              │
│ - "딱 맞는 사료 보기"    │
│ - 추천 상품 미리보기     │
│ - 추천 받기 버튼         │
│                         │
│ [상태 신호 카드들]       │
│ - 가격 신호              │
│ - 소진 신호              │
│ - 건강 신호              │
│                         │
│ [혜택 카드]              │
│ - 포인트 정보            │
│ - 미션 링크              │
│                         │
└─────────────────────────┘
```

**주요 섹션**:

#### 1. 상단 탭바
- **컨텐츠**: "안녕하세요, {닉네임}님!" 또는 "헤이제노"
- **기능**: 사용자 인사말 표시

#### 2. 펫 프로필 요약 카드
- **컨텐츠**:
  - 펫 아바타 (사진 또는 기본 아이콘)
  - 펫 이름
  - 종류, 나이, 체중
  - 건강 고민 태그 (최대 2개)
  - 알레르기 정보
  - "프로필 수정" 링크
- **기능**:
  - 펫 정보 요약 표시
  - 프로필 수정 화면으로 이동
  - 펫 전환 가능 (여러 펫인 경우)

#### 3. 현재 급여 사료 카드
- **컨텐츠**:
  - 사료명
  - 현재 최저가
  - 가격 변동률
  - "현재 급여 중" 배지
  - 소진 예상 정보
- **기능**:
  - 현재 급여 중인 사료 정보 표시
  - 가격 알림 설정
  - 사료 변경

#### 4. 추천 카드
- **컨텐츠**:
  - "우리 애에게 딱 맞는 사료 찾아보기" 타이틀
  - 추천 상품 미리보기 (최대 2개)
  - "지금 추천받기" 버튼
  - 펼쳐진 상태: 상품 상세 정보 + "왜 이 제품일까요?" 섹션
- **기능**:
  - 추천 받기 버튼 클릭 시 추천 애니메이션 화면으로 이동
  - 추천 결과 펼치기/접기
  - 최근 추천이 있으면 바로 표시

#### 5. 상태 신호 카드들 (조건부)
- **컨텐츠**:
  - 가격 신호: "지금 먹는 사료가 최근 30일 중 가장 싸요"
  - 소진 신호: "3일 뒤면 사료가 떨어질 수 있어요"
  - 건강 신호: "이 사료, {펫이름} 관절 고민엔 조금 아쉬울 수 있어요"
- **기능**: 중요한 상태 정보 알림

#### 6. 혜택 카드
- **컨텐츠**: 포인트 정보, 미션 링크
- **기능**: 혜택 화면으로 이동

**상태별 화면**:
- **로딩 상태**: 로딩 스피너
- **펫 없음 상태**: "펫 정보를 추가해주세요" + 프로필 만들기 버튼
- **에러 상태**: 에러 메시지 + 다시 시도 버튼
- **빈 추천 결과**: 사용자 친화적 메시지 + 펫 정보 확인하기 버튼

### 2.2 찜한 사료 화면 (`WatchScreen`)
**경로**: `/watch`  
**파일**: `frontend/lib/features/watch/presentation/screens/watch_screen.dart`

**와이어프레임**:
```
┌─────────────────────────┐
│ 찜한 사료               │
├─────────────────────────┤
│                         │
│ [정렬 필터]              │
│ 저렴한 순 | 가격 변동... │
│                         │
│ ┌─────┐ ┌─────┐         │
│ │상품1│ │상품2│         │
│ │이미지│ │이미지│         │
│ │브랜드│ │브랜드│         │
│ │상품명│ │상품명│         │
│ │가격 │ │가격 │         │
│ │알림 │ │알림 │         │
│ └─────┘ └─────┘         │
│                         │
│ ┌─────┐ ┌─────┐         │
│ │상품3│ │ [+ ]│         │
│ └─────┘ └─────┘         │
│                         │
└─────────────────────────┘
```

**주요 섹션**:

#### 1. 상단 탭바
- **컨텐츠**: "찜한 사료"
- **기능**: 화면 제목

#### 2. 정렬 필터
- **컨텐츠**: 정렬 옵션 칩
  - "저렴한 순"
  - "가격 변동 낮은 순"
  - "최근 추가한 순"
- **기능**: 찜한 상품 정렬

#### 3. 상품 그리드 (2열)
- **컨텐츠**: 각 상품 카드
  - 상품 이미지
  - 브랜드명
  - 상품명
  - 현재 가격
  - 가격 변동률
  - 알림 켜짐/꺼짐 상태
  - 하트 아이콘 (찜 해제)
- **기능**:
  - 찜한 상품 목록 표시
  - 알림 설정/해제
  - 찜 해제
  - 상품 상세 화면으로 이동

#### 4. 상품 추가 카드
- **컨텐츠**: "+" 아이콘 + "상품 추가하기" 텍스트
- **기능**: 마켓 화면으로 이동하여 상품 추가

**상태별 화면**:
- **로딩 상태**: 로딩 스피너
- **에러 상태**: 에러 메시지 + 다시 시도 버튼
- **빈 상태**: 상품 추가 카드만 표시

```
│ [🔔] 알림 설정하고       │
```

**주요 섹션**:

#### 1. 상단 탭바
- **컨텐츠**: "혜택"
- **기능**: 화면 제목

#### 2. 내 포인트 카드
- **컨텐츠**:
  - "내 포인트" 헤더 + 작은 P 아이콘
  - 큰 P 아이콘 + 총 포인트 숫자 (검은색)
  - "XXP 더 받을 수 있어요" 텍스트
- **기능**: 현재 포인트 및 추가 가능 포인트 표시

#### 3. 미션 완료하고 포인트 받기 카드
- **컨텐츠**: 미션 리스트 (하나의 카드에 통합)
  - 각 미션:
    - 아이콘 (Material Icons)
    - 미션 제목 + 포인트 (예: "오늘 추천 사료 찜하고 50P 받기")
    - 진행 상태 (완료/미완료)
    - 화살표 아이콘
  - 미션 사이에 구분선 (Divider)
- **기능**:
  - 미션 목록 표시
  - 미션 클릭 시 해당 기능으로 이동
  - 완료된 미션 표시
  - 포인트 적립 안내

**미션 종류**:
1. 오늘 추천 사료 찜하기 (50P)
2. 알림 설정하기 (30P)
3. 프로필 완성하기 (20P)
4. 리뷰 작성하기 (40P)
5. 친구 초대하기 (100P)
6. 기타 미션들

**상태별 화면**:
- **로딩 상태**: 로딩 스피너
- **에러 상태**: 에러 메시지 + 다시 시도 버튼

### 2.5 더보기 화면 (`MyScreen`)
**경로**: `/me`  
**파일**: `frontend/lib/features/me/presentation/screens/my_screen.dart`

**와이어프레임**:
```
┌─────────────────────────┐
```

## 📝 주요 기능 상세

- **미션 완료**: 포인트 적립
- **포인트 표시**: 현재 포인트 및 추가 가능 포인트
- **미션 진행률**: 완료된 미션 수 표시

- **가격 그래프**: 7일간 가격 추이
- **가격 비교**: 평균 대비 가격 비교
- **알림 설정**: 목표 가격 도달 시 알림
